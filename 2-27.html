Django Messages Framework

為了在編輯或是新增貼文後跳出訊息，需要「from django.contrib import messages」。

在適當的地方加入：
「messages.success(request, "Successfully created ~~~")」或是
「messages.error(request, "Not successfully created ~~~")」或是
「messages.success(request, "<a href='xxx'>Item</a> Saved ~~~",extra_tags='html_safe')」。

另外在編輯或新增完後會跳到該項目的頁面，所以頁面的 html 檔(post_detail)也要編寫以顯示設定的 messages。

{% if messages %}
<ul class="messages">
  {% for message in messages %}
  <li{% if message.tags %} class="{{ message.tags }}"{% endif %}> {% if "html_safe" in message.tags %}{{ message|safe}} {% else %}{{ message }}{% endif %}</li>
  {% endfor %}
</ul>
{% endif %}
