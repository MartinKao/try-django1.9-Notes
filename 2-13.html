Customize Admin

想要更多的功能請參考這個 reference。

「list_display = ["title","updated","timestamp"]」設定要顯示在 admin 畫面的變數。

「list_display_links = ["updated"]」設定有哪些變數在畫面上具有連結到 post 的功能。

「list_filter = ["updated","timestamp"]」設定有哪些變數可以做為排序的依據。

「search_fields = ["title","content"]」設定有哪些變數內容可以被搜尋。

「list_editable = ["title"]」設定在 Post list 畫面中哪些參數可以直接被編輯。不過該參數不能同時設定 links 功能。

以上功能會被實作在 admin.py 作為一個 class，「class PostModelAdmin(admin.ModelAdmin):」。

如果這個 class 適用在 Post class 的管理的話，class 內部要加上一段「class Meta: model = Post」註明是給 Post 用的。
