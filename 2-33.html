File Uploads with FileField and ImageField

在 Post 這個 class 加入「image = models.FileField(null=True, blank=True)」。

每次對 model 修改，都要執行「python manage.py makemigrations」然後「migrate」。

修改 urls.py 以及 settings.py，加入 MEDIA 相關的連結。

在 migrations 資料夾的 forms.py 加入 image 變數。

修改 post_form.html 以接收圖片檔案。

修改 post_list.html 和 post_detail.html 以顯示圖片。

修改 views.py 裡會接收 image 的 function。

要改為 ImageField 需要「pip install pillow」。
