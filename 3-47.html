Implement Django Pagedown for Stack Overflow Style Markdown

使用 django-pagedown，參考此 Django Pagedown 頁面。

可以在 forms.py 改變 publish 的 form 內容形式。
  「publish = forms.DateField(widget=forms.SelectDateWidget)」。

也可以改變 content 的 form 形式。用 Pagedown。

  1. 在 base.html 新增「{% block head_extra %}{% endblock head_extra %}」。
  2. 在 post_form.html 新增「{% block head_extra %}{{ form.media }}{% endblock head_extra %}」。
  3. 在 forms.py 新增「from pagedown.widgets import PagedownWidget」
    以及「content = forms.CharField(widget=PagedownWidget)」。

在 post_list.html 使用之前寫的 content-markdown 這個 class，一樣可以 render。
