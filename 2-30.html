Setup Static Files - CSS & Javascript & Images in Django

關於 static file 可以查看： Managing static files。

在 urls.py 新增：
  「from django.conf import settings」
  「from django.conf.urls.static import static」
以及設定 settings.DEBUG。

在 settings.py 新增兩個設定：「STATICFILE_DIRS」「STATIC_ROOT」。

在虛擬環境新增資料夾「static_cdn」，在 src 新增資料夾「static」。

以上設定完成後，執行「python manage.py collectstatic」，static_cdn 裡的環境就會被設定好。

在 static 資料夾裡建立 css 資料夾，新增 base.css 檔案。

當 static 資料夾做出變動，就要執行「python manage.py collectstatic」，使得 static_cdn 的環境更新。

base.html 需要引用 css檔「<link rel='stylesheet' href='/static/css/base.css' />」。

新增設定「MEDIA_ROOT」以及 media_cdn ，儲存使用者上傳資料。

上面的路徑是寫死的，加入
  「{% load staticfiles %}」以及修改
  「<link rel='stylesheet' href='{% static "css/base.css" %}' />」，
就可以變更 STATIC_URL 的名稱而不影響實際位置。
