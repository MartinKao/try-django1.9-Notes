Versions and Install

「django-admin.py startproject tryjango19」。啟動 Django 專案。

按 ls 會看到專案名稱的資料夾。

接著進入專案資料夾，會看到 manage.py 和一個同樣名稱的資料夾。

執行「python manage.py runserver」。

會看到錯誤訊息不過先不用理它，以及底下出現一段網址。複製網址貼上瀏覽器，會出現 it worked! 的頁面。

「python manage.py runserver 8888」會將 port 換到 8888。預設是 http://127.0.0.1:8000/

按「contrpl + c」離開 server。
